# Preset avatars

`avatar-1.png` … `avatar-10.png` are memoji-style 2D avatars vendored from
[alohe/avatars](https://github.com/alohe/avatars) (the `memo` set), MIT licensed
(© 2022 Alohe). Selected via Ustawienia → Personalizacja; the choice is stored
in `profiles.settings.avatarPresetId` and resolved by `src/lib/theme/avatar-presets.ts`.

To swap an avatar, replace the PNG keeping the same filename.
